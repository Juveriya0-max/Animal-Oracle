# 🐾 Animal Oracle

An interactive animal guessing game where the AI asks questions to guess your animal!

## 🎮 How to Play

1. Click **"Start Game"**
2. Think of an animal
3. Answer 4 questions (Yes/No)
4. Watch the Oracle guess your animal!

## 🧠 Features

- 🤖 AI that learns from you
- 💾 Saves animals in your browser
- 📚 Fun facts about each animal
- 🎨 Playful design

## 🛠️ Technologies

- HTML
- CSS
- JavaScript
